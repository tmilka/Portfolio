Arcade Box

4 Spiele
Python

Retro-Gaming pur

Die Arcade Box ist eine Python-basierte Spielesammlung, die den Charme klassischer Arcade-Automaten direkt in dein Terminal bringt. Über ein intuitives Interface kannst du aus einer Vielzahl von Klassikern wählen, die sowohl Glück als auch Strategie fordern.

Enthaltene Spiele

  • • Russian Roulette: Ein nervenaufreibendes Glücksspiel um das Überleben.
  • • Rock Paper Scissors: Der zeitlose Klassiker gegen die KI.
  • • Guess the Number: Ein Test für deine Intuition und Logik.
  • • Tic Tac Toe: Strategie-Duelle gegen Freunde oder den Computer.

Hinter den Kulissen

Das Projekt wurde entwickelt, um die Grundlagen der Python-Programmierung und algorithmische Zufallsmechaniken zu vertiefen. Ein integriertes Scoreboard nutzt Datei-Handling, um deine Fortschritte über verschiedene Sessions hinweg dauerhaft zu speichern.

Used Tech: Python // Randomization Algos // File I/O
